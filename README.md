# Flipkart Selenium Automation Project

## Project Description
This project automates the basic shopping flow of Flipkart using Selenium WebDriver.

## Scenario Automated
1. Open browser
2. Go to Flipkart website
3. Search for a product
4. Open the product page
5. Click Add to Cart
6. Take screenshot
7. Close browser

## Tools & Technologies
- Java
- Selenium WebDriver
- Maven
- IntelliJ IDEA

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run the main test file

## Output
Screenshots are saved after adding product to cart.
